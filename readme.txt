---- How to build ---

1. Instal JDK1.6.X or later.
2. Install Maven3.0.4 or later.
3. type next commnad.
   > mvn clean package
4. If build success, target/batchfile/demo-distributio.zip will be created.

--- How to run ---
1. Unzip demo-distributio.zip
2. move ./output directory
3. Type next command
  outout> java -jar java -jar prime-1.0-SNAPSHOT.jar
4. Input number you want to get prime.
